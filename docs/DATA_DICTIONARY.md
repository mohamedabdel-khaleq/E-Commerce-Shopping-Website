# 📚 قاموس البيانات

يصف هذا المستند هياكل البيانات الرئيسية المستخدمة في مشروع التجارة الإلكترونية، وخاصة تلك التي يتم تخزينها في `localStorage`.

## 👤 المستخدم (User)

يتم تخزين كائنات المستخدم في `localStorage` تحت المفتاح `users` (كقائمة) وتحت المفتاح `currentUser` (للمستخدم الحالي).

| الحقل       | النوع    | الوصف                                                                | مثال               |
| :---------- | :-------- | :------------------------------------------------------------------- | :------------------ |
| `username`  | `string`  | اسم المستخدم الفريد.                                                | `"john_doe"`        |
| `email`     | `string`  | البريد الإلكتروني للمستخدم.                                         | `"john@example.com"`|
| `password`  | `string`  | كلمة المرور المشفرة (مخزنة كنص عادي لأغراض العرض التوضيحي).          | `"hashed_password"` |
| `isAdmin`   | `boolean` | يشير إلى ما إذا كان المستخدم مسؤولاً أم لا.                          | `true` / `false`    |

## 🛒 سلة التسوق (Cart Item)

يتم تخزين عناصر سلة التسوق في `localStorage` تحت المفتاح `cart`.

| الحقل       | النوع    | الوصف                                                                | مثال                                 |
| :---------- | :-------- | :------------------------------------------------------------------- | :------------------------------------ |
| `id`        | `number`  | معرف المنتج.                                                        | `12345`                             |
| `title`     | `string`  | اسم المنتج.                                                         | `"حقيبة ظهر"`                      |
| `price`     | `number`  | سعر الوحدة للمنتج.                                                  | `109.95`                            |
| `image`     | `string`  | رابط صورة المنتج.                                                   | `"https://example.com/image.jpg"` |
| `quantity`  | `number`  | كمية المنتج في سلة التسوق.                                          | `1`                                 |

## 📦 الطلب (Order)

يتم تخزين كائنات الطلب في `localStorage` تحت المفتاح `orders`.

| الحقل       | النوع    | الوصف                                                                | مثال                                     |
| :---------- | :-------- | :------------------------------------------------------------------- | :---------------------------------------- |
| `id`        | `string`  | معرف فريد للطلب (مثال: `ORD-1678888888888`).                       | `"ORD-1678888888888"`                     |
| `user`      | `string`  | اسم المستخدم الذي قام بالطلب.                                       | `"john_doe"`                              |
| `items`     | `array`   | قائمة بعناصر سلة التسوق (`Cart Item`) التي يتضمنها الطلب.           | `[{ id: 1, title: "...", quantity: 1 }]` |
| `total`     | `string`  | إجمالي سعر الطلب بعد تطبيق أي خصومات (كسلسلة نصية لتنسيق العملة). | `"109.95"`                                |
| `date`      | `string`  | تاريخ ووقت إنشاء الطلب.                                             | `"3/15/2023, 10:30:00 AM"`                |

## 🛍️ المنتج (Product)

يتم جلب بيانات المنتجات من [FakeStoreAPI](https://fakestoreapi.com/) ويتم تخزينها مؤقتًا في `localStorage` تحت المفتاح `products`.

| الحقل         | النوع    | الوصف                                                                | مثال                                 |
| :------------ | :-------- | :------------------------------------------------------------------- | :------------------------------------ |
| `id`          | `number`  | معرف المنتج الفريد.                                                 | `1`                                 |
| `title`       | `string`  | اسم المنتج.                                                         | `"Fjallraven - Foldsack No. 1 Backpack, Fits 15 Laptops"` |
| `price`       | `number`  | سعر المنتج.                                                         | `109.95`                            |
| `description` | `string`  | وصف تفصيلي للمنتج.                                                  | `"Your perfect pack for everyday use..."` |
| `category`    | `string`  | فئة المنتج.                                                         | `"men's clothing"`                |
| `image`       | `string`  | رابط صورة المنتج.                                                   | `"https://fakestoreapi.com/img/81fPKd-2s-L._AC_SL1500_.jpg"` |
| `rating`      | `object`  | تقييم المنتج.                                                       | `{ "rate": 3.9, "count": 120 }`   |
| `rating.rate` | `number`  | متوسط التقييم.                                                      | `3.9`                               |
| `rating.count`| `number`  | عدد التقييمات.                                                      | `120`                               |
